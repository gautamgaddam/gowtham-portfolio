# Batman Dark Product UI

Use this skill when styling product screens in the Batman-dark direction.

## Visual Language

- Base: near-black, charcoal, deep navy.
- Primary accent: neon green for action/success.
- Secondary accent: electric blue for selected tools and cockpit panels.
- Caution: amber. Danger: red only.
- Keep gradients subtle and functional.
- Avoid decorative blobs/orbs and noisy nested cards.

## Component Rules

- Use compact command bars, segmented controls, status chips, and cockpit panels.
- Keep borders thin and purposeful.
- Use glow sparingly for active states and focus areas.
- Prefer dense but readable layouts for repeated operational use.
