# Health Accessibility

Use this skill for health UI changes that affect readability or interaction.

## Rules

- Maintain readable contrast on dark surfaces.
- Provide labels or titles for icon-only controls.
- Keep keyboard-reachable controls for chat, tabs, drawer, avatar actions, and panel resize buttons.
- Avoid relying on color alone for safety/evidence states.
- Do not obscure medical disclaimers, errors, or save failures.
- Support mobile without horizontal scrolling.

## Verification

- Check tab order for primary controls.
- Check focus visibility.
- Check long text in buttons/chips does not overflow.
